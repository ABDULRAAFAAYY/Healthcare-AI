Project Abstract

Healthcare AI is a cutting-edge web application designed to assist medical professionals in early disease detection. By combining Natural Language Processing (NLP) for symptom analysis and Computer Vision (Vision Transformers) for medical imaging, the system provides accurate, real-time diagnostic support.

Pneumonia Image Detector

Vision Transformer (ViT)

Utilizes the nickmuchi/vit-finetuned-chest-xray-pneumonia model from Hugging Face.

  • Input: Chest X-Ray Images (JPG/PNG)
  • Architecture: Vision Transformer (ViT)
  • Classification: Normal vs. Pneumonia
  • Features: Confidence scores, severity assessment, and medical recommendations.

Symptom Predictor

NLP Analysis

Intelligent symptom mapping and disease probability calculation.

  • Input: Natural language symptoms
  • Logic: Probabilistic mapping engine
  • Output: Potential diseases with likelihood percentages
  • Database: Comprehensive medical knowledge base
